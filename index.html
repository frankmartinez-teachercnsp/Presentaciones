1. El Sistema Internacional de Unidades (SI)

El Sistema Internacional (SI) es el sistema de unidades estándar utilizado en casi todo el mundo para la ciencia, la industria y el comercio. Proporciona una base coherente para las mediciones, asegurando que los resultados sean comprensibles y reproducibles globalmente.

Se basa en siete unidades fundamentales:

Magnitud Física Unidad Base Símbolo Ejemplo Común
LongitudMetromLa altura de una puerta (aprox. 2 m)
MasaKilogramokgUna bolsa de azúcar (1 kg)
TiempoSegundosLa duración de un parpadeo (aprox. 0.3 s)
Corriente EléctricaAmperioACorriente de un cargador de móvil (1-2 A)
TemperaturaKelvinKTemperatura ambiente (aprox. 293 K)
Cantidad de SustanciaMolmol18g de agua es aproximadamente 1 mol
Intensidad LuminosaCandelacdLa luz de una vela (aprox. 1 cd)

A partir de estas unidades base, se derivan otras unidades como el Newton (N) para la fuerza o el Pascal (Pa) para la presión.

2. Notación Científica

La notación científica es una forma de escribir números muy grandes o muy pequeños de manera compacta. Se expresa como un número entre 1 y 10 multiplicado por una potencia de 10.

Formato: $a \times 10^n$

  • a: Es un número mayor o igual que 1 y menor que 10 (el coeficiente).
  • n: Es un número entero (el exponente).

Ejemplos:

Números Grandes

La distancia de la Tierra al Sol es aproximadamente 149,600,000,000 metros.

1.496 x 1011 m

(El punto decimal se movió 11 lugares a la izquierda).

Números Pequeños

El diámetro de un átomo de hidrógeno es aproximadamente 0.000000000106 metros.

1.06 x 10-10 m

(El punto decimal se movió 10 lugares a la derecha).

3. Cifras Significativas

Las cifras significativas de un número son los dígitos que aportan información real sobre la precisión de una medida. Indican la certeza de un valor.

Reglas para determinarlas:

  • Los dígitos distintos de cero siempre son significativos. (Ej: 123 tiene 3 c.s.)
  • Los ceros entre dígitos distintos de cero son significativos. (Ej: 101.5 tiene 4 c.s.)
  • Los ceros a la izquierda del primer dígito no nulo no son significativos. (Ej: 0.085 tiene 2 c.s.)
  • Para números con punto decimal, los ceros al final del número son significativos. (Ej: 2.500 tiene 4 c.s.)
  • Para números sin punto decimal, los ceros finales pueden ser ambiguos. Se usa notación científica para aclarar. (Ej: 300 puede tener 1, 2 o 3 c.s. Se escribe 3 x 102 (1 c.s.), 3.0 x 102 (2 c.s.), o 3.00 x 102 (3 c.s.)).

En cálculos:

  • Suma y Resta: El resultado se redondea al mismo número de decimales que el término con menos decimales. (Ej: 12.11 + 18.0 + 1.013 = 31.123 → 31.1)
  • Multiplicación y División: El resultado se redondea al mismo número de cifras significativas que el factor con menos cifras significativas. (Ej: 4.56 × 1.4 = 6.384 → 6.4)

4. Representación de Datos en Gráficas

Las gráficas son herramientas visuales poderosas para analizar datos, identificar patrones y comunicar resultados de manera efectiva.

Componentes Clave de una Gráfica:

  • Título: Describe claramente de qué trata la gráfica.
  • Eje X (Abscisa): Representa la variable independiente. Debe tener una etiqueta y unidades.
  • Eje Y (Ordenada): Representa la variable dependiente. Debe tener una etiqueta y unidades.
  • Escala: Los intervalos en los ejes deben ser consistentes y apropiados para los datos.
  • Puntos de Datos: Marcas que representan las mediciones.
  • Línea de Tendencia: Una línea (recta o curva) que muestra la relación general entre las variables.
